# Aprendendo a Contar 🎮

Um jogo educativo desenvolvido em **C#** para ajudar crianças do ensino fundamental a praticar e aprender contagem de forma interativa e divertida.

## 📋 Sobre o Projeto

Este projeto foi criado com o objetivo de tornar o aprendizado de matemática básica mais acessível e atrativo para crianças em idade escolar. O jogo oferece uma abordagem lúdica para o ensino de contagem.

## 🎯 Objetivo

Fornecer uma ferramenta educativa que:
- Facilite o aprendizado de contagem
- Torne o processo divertido e engajador
- Seja adequado para crianças no ensino fundamental (primário)

## 🛠️ Tecnologias

- **C#** - Linguagem de programação principal
